## 📋 라우팅 구조

```mermaid
stateDiagram-v2
    [*] --> Loading
    Loading --> LoginScreen: !user
    Loading --> Dashboard: user exists
    
    LoginScreen --> Dashboard: successful login
    
    Dashboard --> QuizSelection: navigate('quiz-selection')
    Dashboard --> WrongAnswers: navigate('wrong-answers')
    Dashboard --> Analytics: navigate('analytics')
    
    QuizSelection --> QuizScreen: onSelectQuiz(quizData)
    QuizScreen --> QuizResults: onComplete(results)
    QuizResults --> Dashboard: navigate('dashboard')
    QuizResults --> QuizSelection: navigate('quiz-selection')
    
    WrongAnswers --> QuizSelection: practice wrong answers
    Analytics --> QuizSelection: study weak subjects
```
