# Design Assets

이 폴더는 Figma 디자인 시스템에서 추출한 CSS 컴포넌트와 스타일을 관리합니다.

## 폴더 구조

- `components/` - 개별 UI 컴포넌트 스타일
- `layouts/` - 페이지 레이아웃 스타일
- `tokens/` - 디자인 토큰 (색상, 타이포그래피, 간격 등)
- `utilities/` - 유틸리티 클래스
